This project demonstrates how to set widget margins, adjusting widget height
and controlling widget visibility.

# Configure Widget Margins

See the -widgetMarginInsetsForProposedMarginInsets: method in the widget's
TodayViewController.m.


# Adjusting Widget Height

See the -sizeToggleButtonTapped: method in the widget's TodayViewController.m.


# Animating Alongside Height Adjustment

See the -viewWillTransitionToSize: method in the widget's TodayViewController.m.


# Control Widget Visibility

See the -setWidgetVisibility: method in the app's ViewController.m.
